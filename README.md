# Hava Durumu Uygulaması

Python ile hava durumu bilgisi çeken basit bir uygulama.

## Kullanılan Teknolojiler
- Python
- Requests
- Weather API

## Açıklama
Hava durumu uygulaması yaparken kullandığım dili pyhton olarak belirledim. Arayüz kullanma kolaylığı açısından en uygun yazılım dili pyhtondur. Projemde internet sitesi üzerinden hava durumu sitemin ulaşılabilir linkini pythoba çekip bir fonksiyon üzerinden datalarımızı çekip bunu mobil uygulamama yansıtmak için değişkenler ve fonksiyonlar tanımladım. Kullandığım site veri çekim işlemine izin veren bir site olduğu için veriyi çekme konusunda bir problem yaşamadım. Veriyi çektikten sonra site üzerinden sıcaklık, açıklık ve icon gibi değikenleri kendi değişkenlerimize eşitleyip kendi uygulamamız üzerinde kullandım. Bu işlemlerden sonra arayüzümüzü yapmak için Mobil uygulamanın tasarımına geçtim. Büyüklük yazı tonu gibi arayüz işlemlerini yapıp uygulamamızı hazır hale getirdim.
